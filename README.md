# Exercise 2
Use ReactJS to create a simple 2-player OFFLINE caro game.

## How to run:
### `yarn install`
To install required dependencies:
- phosphor-react
- styled-components
- react-draggable
- react-modal

### `yarn start`
Run the app in development mode. 
Open http://localhost:3000 to view it in browser.

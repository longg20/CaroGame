A simple 2-player OFFLINE caro game using ReactJS with Redux-Toolkit to manage states.

## How to run:
### `yarn install`
To install required dependencies:
- phosphor-react
- styled-components
- react-draggable
- react-modal
- redux-toolkit
- lodash

### `yarn start`
Run the app in development mode. 
Open http://localhost:3000 to view it in browser.
